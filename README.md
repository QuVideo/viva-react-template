# VivaVideo React Template

react + antd 脚手架工具模版

# How to use

```bash
# 安装依赖
npm install
# 开发
npm start
# 构建
npm run build
# 校验代码
npm run lint
```

# 实现功能

- [x] Hot Module Replacement
- [x] Babel for React & ES6
- [x] ESLint checkout
- [x] 样式处理
- [x] 图片处理
- [x] JS 压缩
- [x] CSS 分离
- [x] 模块化 CSS
- [x] SourceMap 代码映射
- [x] 开发环境 Dashboard 界面
